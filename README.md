# MindLock Website

**WEBSITE FILES** - This folder contains the landing page for MindLock, separate from the iOS app.

## Files

- `index.html` - Main landing page HTML
- `styles.css` - Website styling and responsive design
- `script.js` - Website interactivity and animations
- `README.md` - This file

## Purpose

This website serves as the official landing page for MindLock, explaining the app's concept and value proposition. It's designed to:

1. **Explain the concept** - Turn screen time slips into charitable impact
2. **Show how it works** - Next-day accountability with charitable donations
3. **Build credibility** - Professional presentation for Apple Developer approval
4. **Collect interest** - Waitlist signup for launch

## Design

- **Modern, clean aesthetic** with gradient accents
- **Mobile-responsive** design
- **Interactive elements** with smooth animations
- **Professional presentation** suitable for Apple Developer portal

## Usage

Simply open `index.html` in a web browser to view the landing page locally, or deploy to any web hosting service.

---

**Note**: These are website files, not iOS app files. The iOS app is located in the `ios/` folder.
